Allgemeine Informationen

Hinweis: Dieses Plugin ist freie Open-Source-Software. Es wird keine Haftung für Schäden oder Fehlfunktionen übernommen.

Voraussetzungen:
  • Benutzer müssen einen Telegram-Benutzernamen besitzen, um sich via SSO anzumelden.
  • Ein Telegram-Bot-Token ist erforderlich, um die Authentifizierung durchzuführen.
  • Eine öffentlich erreichbare Domain mit gültigem SSL-Zertifikat wird für das Login-Widget benötigt.
  • Den Bot-Token erhältst du beim offiziellen Bot @BotFather.
  • Verwende den Befehl /setdomain im Chat mit @BotFather, um deine Jellyfin-Domain für den Bot freizugeben.
  • Ein Telegram-Benutzer muss Jellyfin-Administrator sein oder mindestens einer berechtigten Telegram-Gruppe angehören, um sich anzumelden.
Kurzanleitung:
  1. Trage unten dein Telegram-Bot-Token ein und klicke auf "Konfiguration speichern".
  2. Erstelle eine Gruppe im Bereich "Gruppenverwaltung" (z.B. "Freunde") und klicke auf "Hinzufügen".
  3. Wähle die Gruppe in der Liste aus.
  4. Wähle die Bibliotheken, auf die diese Gruppe Zugriff erhalten soll.
  5. Trage die Telegram-Benutzernamen der Mitglieder ein (einer pro Zeile).
  6. Speichere die Gruppen über "Gruppen speichern".
